# Development setup

> yarn is required

After cloning this repo:
### Website
  - Run ```yarn start``` 
  - Go back to the root folder
  - Run ```yarn dev:web``` to up the project

# Tests
### Server
  - Run ```yarn test:server``` at the root folder
